# Reading_label
this tool is used for label the classifier of passages

data:原文本文件,一个txt一篇阅读

output:标注后的文件夹

app->label_data:类别名与ID号的对应

安装:flask

安装:flask_script

运行:

python manage.py runserver

浏览器打开

http://127.0.0.1:5000/reading_label/index
